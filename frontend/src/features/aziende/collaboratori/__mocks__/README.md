# Mocks UI per test manuali e2e

In questo folder è possibile aggiungere, se necessario, mock di dati o configurazioni specifiche per consentire test manuali o automatizzati della UI senza dipendenze esterne.

Per il Task 1080 non sono stati introdotti nuovi componenti UI, ma solo test; i data-testid utilizzati nei test Playwright fanno riferimento a selettori già previsti e usati nei task precedenti per la gestione collaboratori.
