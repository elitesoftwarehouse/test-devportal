Modello dati - Registrazione utenti esterni

Questa pagina documenta i campi minimi previsti a livello di modello dati per la registrazione di professionisti esterni singoli e aziende fornitrici. Non rappresenta una UI definitiva ma una vista di supporto per analisi tecnica.

ExternalUser (professionista / utente azienda)

  • email (obbligatorio, univoco fra gli esterni)
  • passwordHash (obbligatorio)
  • subjectType (EXTERNAL_PROFESSIONAL | EXTERNAL_COMPANY)
  • firstName (obbligatorio se EXTERNAL_PROFESSIONAL)
  • lastName (obbligatorio se EXTERNAL_PROFESSIONAL)
  • codiceFiscale (opzionale per professionista)
  • partitaIva (opzionale per professionista)

  • tosConsented (obbligatorio)
